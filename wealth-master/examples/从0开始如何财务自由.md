## 阶段定位

用户处于**阶段一（0~20万）**。

这个阶段最大的风险不是投资亏损，而是：
- 收入太低
- 没有储蓄能力
- 没有可出售的技能

```
20万本金，年化10% = 一年赚2万。
工资从10万涨到20万 = 一年多赚10万。

在20万以前，提升收入比研究投资重要5倍。
```

## 核心建议

### 方向一：主业深挖

有稳定工资是很好的基础。问自己：

> 你的技能在你的行业里，是不是可出售、可涨价的？

如果是——深耕，争取1~2年内涨薪/跳槽。如果不确定——这是你现在最该花时间的地方。

2026年的杠杆技能：编程/AI工具链/写作/销售。

### 方向二：定投，别碰别的

每周只想花0~1小时——完全自动化：

```
每月发工资那天，自动买入 → 沪深300 ETF（510300）
比例：90% 股票 + 10% 现金（3~6个月生活费放货币基金）

不要做：
❌ 研究个股
❌ 追热点
❌ 刷雪球看K线
❌ 碰加密货币
❌ 有任何"搏一把"的念头

操作频率：每月1次，每次5分钟。
```

### 方向三：学芒格

| 思维 | 对阶段的用处 |
|------|-------------|
| 反过来想 | 先想"怎么保证财务不自由"——不储蓄、月光、不学新技能、追涨杀跌——然后避开这些事 |
| 能力圈 | 你不懂的就是不懂。沪深300在你能力圈内（你理解"中国经济的整体增长"） |

## 一句话总结

> 你现在最重要的资产不是20万本金，是你20年后还有的赚钱能力和复利时间。
