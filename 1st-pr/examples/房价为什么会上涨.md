## 第一性原理分析：为什么房价会上涨？

### 1. 问题定义

**核心问题**：为什么在大多数城市，房价在长期内（10年以上）呈现上涨趋势？

**范围边界**：
- 在范围内：长期趋势、实际价格上涨（扣除通胀）、城市住房
- 不在范围内：短期周期波动、名义价格上涨、农业/偏远地区

**成功标准**：区分房价的消费品属性与资产属性，识别不可改变的底层驱动力。

### 2. 拆解至基本元素

| 维度 | 基本事实 | 不可改变性 |
|------|---------|-----------|
| 价格定义 | 价格 = 供需交点 | ✅ 经济学公理 |
| 土地供给 | 好地段土地总量固定，位置不可复制 | ✅ 物理约束 |
| 位置价值 | 核心城市生产率 > 全国平均 → 高收入聚集 | ✅ 经济地理事实 |
| 信贷本质 | 房贷 = 未来收入提前支用 | ✅ 金融定义 |
| 折现定价 | 资产价格 = 未来租金折现和：P = ΣRₜ/(1+r)ᵗ | ✅ 金融数学 |
| 资产属性 | 房子同时是消费品（居住）+ 资本品（投资） | ✅ 双重属性 |

### 3. 假设挑战

| 假设 | 挑战 | 判定 |
|------|------|------|
| "人口增长所以房价涨" | 日本人口下降15年，东京房价仍在涨 | ❌ 不成立 |
| "土地稀缺所以房价涨" | 城市周边大量空地，稀缺的是已配套的好地段 | ⚠️ 需要松动 |
| "通货膨胀所以房价涨" | 通胀解释名义价格，不能解释"房价比通胀涨得快" | ❌ 混为一谈 |
| "建筑材料涨价" | 建筑成本占房价20-40%（一线更低），涨幅远慢于房价 | ❌ 占比太小 |
| "供不应求" | 是现象描述，不是根因 | ⚠️ 方向对但不够深 |

### 4. 从零重建

**推理链 1：核心城市的结构性错配**

```
核心城市生产率高 → 高收入人群聚集
  → 购房预算 = 收入 × 信贷倍数
  → 信贷扩张时：利率下降 → 同一群人的购买力大幅上升
  → 但核心地段供给无法快速增加（位置固定 + 审批周期长）
  → 增量购买力被有限供给吸收
  → 结果不是更多人买到房，而是价格上升
```

**推理链 2：折现率下行驱动的资产升值**

```
房价 = 未来租金的折现和
  → 当 r（折现率≈无风险利率）持续下降
  → 同样的未来租金流，现值上升
  → 房价上涨
  → 过去40年全球利率持续下行 → 系统性的助推器
```

**推理链 3：信贷的正反馈螺旋**

```
房价↑ → 房主财富↑ → 可抵押品价值↑
  → 银行更愿意放贷 → 信贷扩张
  → 购房者购买力↑ → 需求↑ → 房价↑
```

**三个层次答案**：

| 层次 | 核心机制 | 一句话 |
|------|---------|--------|
| 第一层：最直接驱动 | 生产率优势+土地供给刚性 | 好地方越来越值钱，但好地方不增加 |
| 第二层：信用放大器 | 利率下降+信贷扩张 | 不是人更有钱了，是能借更多了 |
| 第三层：反馈螺旋 | 上涨→抵押↑→信贷↑→上涨 | 房地产自带的加速器 |

### 5. 验证计划

- [ ] 跨国面板数据验证：利率下降幅度大的国家，房价涨幅更大
- [ ] 城市间对比：生产率差距大的城市对（如深圳vs鹤岗），房价差距随时间拉大
- [ ] 信贷周期与房价周期的相关性验证
- [ ] 证伪测试：如果房价上涨只是"人口/通胀/成本"驱动，人口下降的国家房价应普跌——但东京是反例

### 6. 结论

**核心洞察**：房价上涨不是单一原因，而是三个层次因素叠加的结果。最根本的是"好地段土地供给刚性 + 该地段收入增长"的结构性错配，信贷扩张放大这个效应，正反馈螺旋让它自己加速自己。大部分流行的解释（人口、通胀、成本）要么只看表层，要么搞错了因果方向。

**已知权衡**：短期政策干预（限购、限贷）可以抑制但无法消除结构性驱动；不同国家的制度差异会影响传导路径和幅度。
