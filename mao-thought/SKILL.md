## When to invoke this skill

Proactively invoke this skill when:
- The user asks for systemic analysis of a complex problem ("帮我分析一下...", "为什么...", "怎么做...")
- A problem has multiple interlocking factors that need to be untangled
- The user is stuck and needs a structured thinking framework
- A decision involves trade-offs that need systematic evaluation
- The user wants a phased plan rather than a one-shot answer

Natural language triggers (Chinese):
- 帮我分析一下 / 用毛泽东思想分析 / 客观分析 / 系统分析
- 为什么...（追问根因类问题）
- 怎么办...（寻求解决方案类问题）
- 主要矛盾是什么 / 怎么抓重点 / 该砍掉什么

# 毛泽东思想分析框架 /mao-thought

## 定位

这不是一个历史教育 Skill。这是一个**将毛泽东思想方法论转化为现代问题分析工具**的思维框架。

适用于：产品设计、技术架构、商业决策、职业规划、项目管理、个人成长——任何需要系统性深度分析的领域。

## 核心分析流程（五步法）

当收到一个分析任务时，按以下步骤展开：

### 第一步：实事求是——搞清事实

> *"没有调查，就没有发言权。"* ——《反对本本主义》

**动作**：
- 列出已知的客观事实（不是观点，是事实）
- 识别缺失的关键信息（必须去调研/确认的）
- 区分"假设"和"已确认"
- 如果信息不足，明确指出并建议调研方式

**输出**：一份事实清单 + 信息盲区标注

### 第二步：矛盾分析——找到主要矛盾

> *"事物的矛盾法则，即对立统一的法则，是唯物辩证法的最根本的法则。"* ——《矛盾论》

**动作**：
- 列出问题中的所有矛盾对（如：深度 vs 广度、速度 vs 质量、成本 vs 收益）
- 识别哪个是**主要矛盾**（解决它，其他矛盾缓解或自然解决）
- 分析矛盾的主要方面（哪一方在当前阶段起主导作用）

**输出**：矛盾列表 + 主要矛盾标注 + 解决方向

### 第三步：持久战分解——分阶段推进

> *"战争将经历战略防御、战略相持、战略反攻三个阶段。"* ——《论持久战》

**动作**：
- 将整体目标分解为 N 个阶段
- 每个阶段定义：
  - 核心目标（只一个）
  - 边界条件（完成的判断标准）
  - 资源需求（时间/金钱/人力）
  - 退出标准（什么情况下放弃或转向）
- 当前阶段是什么？下一阶段是什么？

**输出**：阶段路线图 + 当前阶段行动清单

### 第四步：集中优势兵力——聚焦突破

> *"集中绝对优势兵力，包围歼敌一部，各个歼灭。"* —— 军事原则

**动作**：
- 识别当前阶段最核心的一件事
- 列出"不做"的清单（比"要做"更重要）
- 砍掉所有不直接服务当前阶段目标的东西
- 明确一个可验证的阶段性成果

**输出**：当前唯一重点 + 不做清单 + 验证标准

### 第五步：群众路线——反馈闭环

> *"从群众中来，到群众中去。"*

**动作**：
- 识别这件事的"群众"是谁（用户？客户？团队？）
- 如何最低成本获取他们的真实反馈？
- 反馈如何融入下一轮迭代？

**输出**：反馈获取方案 + 迭代节奏

## 补充分析工具

### 工具 A：战略上藐视，战术上重视

面对艰巨任务时：
- **战略层**：树立必胜信念，看到全局可成
- **战术层**：认真对待每个细节，充分准备每个步骤
- 不可偏废：有战略无战术 = 空谈，有战术无战略 = 迷失

### 工具 B：独立自主

依赖分析三步：
1. 哪些事必须自己做？（核心竞争力不可外包）
2. 哪些事可以借力？（外部资源/工具/合作）
3. 哪些事在中间地带？（有条件地借力）

原则：把方针放在自己力量的基点上。

### 工具 C：批评与自我批评

项目复盘/决策回顾时：
- 哪些判断被证明是正确的？为什么？
- 哪些判断被证明是错误的？根本原因？
- 如果重来一次，会在哪个节点做出不同选择？
- 不找借口，找原因。不追责，追机制。

### 工具 D：波浪式前进

理解事物发展的非线性和规律：
- 进展不是直线，是有起有伏的波浪
- 退一步是为了进两步
- 不应因阶段性挫折否定全局方向
- 留有余地——计划别太满，话别说得太绝

## 知识库（嵌入式参考）

### 一、实事求是

从客观实际出发，理论联系实际，在实践中检验和发展真理。

三层含义：
1. **实事**：客观存在的一切事物 → 先搞清楚事实
2. **是**：事物的内部联系和规律 → 找出本质规律
3. **求**：研究探索 → 主动探究而非被动接受

反对三种倾向：
- 教条主义：照搬本本，不结合实际
- 经验主义：只信自己经历，拒绝理论指导
- 主观主义：从愿望出发，而非从实际出发

方法论工具：
- 调查研究法："调查如十月怀胎，解决问题如一朝分娩"
- 矛盾分析法：承认矛盾普遍性，抓主要矛盾和矛盾主要方面
- 实践检验法：实践→认识→再实践→再认识，循环往复

### 二、矛盾论（核心分析武器）

**承认矛盾的普遍性**：矛盾存在于一切事物的发展过程中，贯穿始终。

**抓主要矛盾**：
- 众多矛盾中，必有一个起主导决定作用的
- 主要矛盾解决→次要矛盾迎刃而解或上升为新主要矛盾
- 所有资源优先投向主要矛盾

**矛盾的主要方面**：
- 矛盾的两方面中，有一方占主导地位
- 认清主导方，判断事物发展走向
- 创造条件转化矛盾的主要方面

**具体问题具体分析**：
- 不同质的矛盾用不同质的方法解决
- 不搞一刀切，不套公式

**区分两类矛盾**：
- 对抗性矛盾（不可调和，需要彻底解决）
- 非对抗性矛盾（可调和，通过协商/调整解决）

**经典原文**：
> "事物发展的根本原因，不是在事物的外部而是在事物的内部，在于事物内部的矛盾性。"

### 三、实践论

认识的全过程：
```
感性认识（表面现象）
    ↓ 飞跃一
理性认识（本质规律）
    ↓ 飞跃二
革命实践（检验真理）
    ↓
新的感性认识 → 循环往复，螺旋上升
```

核心观点：
- 实践是认识的来源、动力、目的和检验标准
- "你要知道梨子的滋味，就要亲口吃一吃"
- 反对纸上谈兵——原型比文档更有说服力
- 但也不能轻视理论——经验要上升为理论才能被复用

### 四、群众路线

公式化表达：
```
群众分散意见 → 集中研究 → 系统化决策 → 回到群众执行 → 根据反馈修正
```

现代语境下的"群众"：
- 产品开发：用户/客户
- 项目管理：团队成员
- 职业发展：同行/mentor/市场
- 任何决策的受影响方

方法论要点：
- **从群众中来**：深入一线，收集真实反馈，甘当小学生
- **到群众中去**：把方案讲清楚，让执行者理解"为什么"
- **一般与个别结合**：先试点再推广，不要一上来就铺开

### 五、持久战思维

三个阶段框架：

| 阶段 | 特征 | 策略 |
|------|------|------|
| 战略防御 | 力量对比处于劣势，资源有限 | 保存实力，建立基础，不求速胜 |
| 战略相持 | 力量趋近平衡，阵地战拉锯 | 积累优势，等待转折，局部突破 |
| 战略反攻 | 力量对比扭转，主动权在手 | 全面出击，速战速决，彻底解决 |

应用要点：
- 每个阶段有明确的目标和边界
- "防御中的进攻，持久中的速决"——即使在防御期也要有局部突破
- 阶段转化需要条件积累，不强行跃进
- "兵民是胜利之本"——根本力量来自群众（用户/团队/社区）

### 六、集中优势兵力

> *"伤其十指不如断其一指"*

操作原则：
1. **聚焦**：同一时期只打一个主战场
2. **绝对优势**：在选定的战场上投入压倒性资源
3. **速决**：选定的目标要能快速拿结果，不要拖成持久战
4. **连续**：打一场胜一场，用胜仗积累 momentum
5. **机动**：主战场选错了就果断切换，不在错误的地方恋战

"不做"清单的价值：
这个原则的核心不是"做什么"，而是**"不做什么"**。每接受一个新任务，意味着放弃十个潜在任务——敢于放弃需要洞察力。

### 七、活的灵魂

三个基本原则的内在关系：

```
实事求是（核心）
    ↓ 决定了
认识路线 + 思想路线
    ↓ 导出了
群众路线（出发点）      独立自主（立足点）
→ 从群众中来获取事实    → 依靠自己力量解决问题
→ 到群众中去验证事实    → 走适合自己的道路
```

三者统一于解决问题的全过程。不可偏废。

## 输出格式规范

分析问题时，使用以下输出结构：

```markdown
## 一、实事求是：事实与盲区
[已知事实]
[信息盲区]
[需要调研的方向]

## 二、矛盾分析：主要矛盾
[矛盾列表，标注主要矛盾]
[矛盾的主要方面]
[解决方向]

## 三、持久战分解：阶段规划
[阶段表格：目标 / 边界 / 资源 / 退出标准]
[当前阶段行动清单]

## 四、集中优势兵力：聚焦与取舍
[当前唯一重点]
[不做清单]

## 五、群众路线：反馈闭环
[反馈获取方案]
[迭代节奏]
```

## 使用原则

1. **以分析为主，引用为辅**：引用原文是为了支持分析，不是为了展示知识。每段引用后必须跟着"这意味什么"。
2. **用现代语言翻译**：把"帝国主义"翻译成"竞争对手"，把"革命"翻译成"变革"，把"群众"翻译成"用户/客户/团队"——让方法论活在当下。
3. **结论要有操作性**：分析完必须有"所以下一步做什么"。不能只有理论没有行动。
4. **承认局限性**：任何分析框架都有盲区。如果问题不适合用毛泽东思想分析，直说，不要强套。
5. **不搞个人崇拜**：用方法论，不用语录轰炸。框架的价值在于逻辑，不在于出处。
