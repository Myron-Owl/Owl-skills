## 诚实前提

**不同财富阶段需要完全不同的策略。** 给一个只有10万的人推荐永久组合，
就像给小学生推荐退休规划——不是错，但这会儿用不上。

你不会在这里听到"买这个股票"。
但你会得到一个清晰的路线图：
**你现在在哪个阶段 → 这个阶段最重要的事是什么 → 对应的策略是什么。**

### 指数投资与主动选股的双轨原则

- **默认轨：指数化投资。** 对没有时间、兴趣或稳定研究能力的用户，优先使用低费率宽基、定投与资产配置。
- **进阶轨：主动选股。** 仅在用户主动要求研究个股，且通过准入检查后，使用彼得·林奇框架作为小比例卫星策略。
- 不把“熟悉产品”当作买入理由；它只能提供研究线索，财务质量、估值与反证检查缺一不可。
- 讨论主动选股或公司分析时，必须读取 [references/peter-lynch-stock-picking.md](references/peter-lynch-stock-picking.md)。

## Stage 0：债务出清 — 先还债，再投资

> **"如果你买了你不需要的东西，很快你就会卖掉你需要的东西。"** — Warren Buffett

在五阶段之前，先回答一个问题：**你有没有负债？** 如果有，必须先用这套框架处理。

### 三区域决策框架

不是所有负债都是坏东西。分清楚：

| 区域 | 利率范围 | 年化8%等价物 | 行动 |
|------|---------|-------------|------|
| 🔴 红区 | > 8% | 信用卡分期、消费贷、网贷、借呗花呗非免息 | **立即停止投资，全力还债** |
| 🟡 黄区 | 4~8% | 车贷、部分信用贷 | **部分还债 + 部分投资**，看个人风险偏好 |
| 🟢 绿区 | < 4% | 房贷、公积金贷款、助学贷款 | **按节奏还，正常投资** |

**核心逻辑：**

```
红区债务年化 15% → 还掉它就等于做了一个无风险年化 15% 的投资
沪深300长期年化 ≈ 8~10%（有风险）
→ 还高息债的确定性收益 > 任何投资的预期收益
```

### 红区债务处理流程

```
Step 1: 立刻停止借新还旧
  → 每一笔新借款都在加深陷阱
  → 除非是找家人/朋友借无息资金置换高息债

Step 2: 砍掉所有非必要支出
  → 外卖、打车、订阅、购物——暂停 3 个月
  → 腾出每一分钱还本金

Step 3: 选择还债策略
  ┌──────────────────────────────┬──────────────────────────────┐
  │  雪球法 (Dave Ramsey)         │  雪崩法 (数学最优)            │
  ├──────────────────────────────┼──────────────────────────────┤
  │ 按金额从小到大还              │ 按利率从高到低还               │
  │ 每还清一笔获得心理胜利         │ 总利息支出最少                 │
  │ 适合：需要正向反馈的人         │ 适合：能理性执行的人            │
  └──────────────────────────────┴──────────────────────────────┘

Step 4: 建立防反弹机制
  → 还清后不要立即恢复消费水平
  → 把"月供"变成"月存"，养成习惯
```

### 家庭债务的特殊性

家庭债务比个人债务复杂得多——因为有**两个人的行为**在影响财务状况。

#### 原则一：透明 > 完美

```
问题：老婆/老公不知道对方欠了多少
  → 先坐下來，把所有债务摊在桌上
  → 列总表：谁欠谁、利率、月供、剩余期限
  → 不看总金额吓自己，看每笔的利率和优先级
```

**"不透明的财务状况本身就是最大的风险。"** 不知道的债务比知道的债务可怕一万倍。

#### 原则二：分开处理，统一调度

| 债务类型 | 谁的责任 | 共同 action |
|---------|---------|------------|
| 共同债务（房贷/装修贷） | 共同承担 | 从家庭收入中统一支付 |
| 个人消费债（一方的高息消费贷） | 个人承担还款责任 | 家庭可以短期输血帮助，但要有还款协议 |
| 人情债（借亲友） | 谁借的谁还 | 优先还清——人情债的隐性利息最高 |

#### 原则三：救急不救懒

如果一方有高息消费贷，另一方帮忙还的前提：
```
✅ 借款人承诺不再新增消费贷
✅ 共同制定还款计划并执行
✅ 控制日常消费，避免"这边还在还债那边继续花"
❌ 不要无底线输血——会养成依赖，甚至鼓励更多消费
```

底线：**"先救婚姻，再救债务。"** 如果因为债务问题每天争吵，先解决沟通问题。钱可以再赚，关系破裂了成本更高。

#### 大师：Dave Ramsey（戴夫·拉姆齐）

> **" Debt is dumb. Cash is king. "**

美国最著名的个人理财/债务咨询专家。他的"婴儿步骤"（Baby Steps）框架帮助了数百万人走出债务泥潭。

**七步法：**
```
Step 1: 存 $1000 应急金（快速缓冲）
Step 2: 用雪球法还清除房贷外的所有债务
Step 3: 存 3~6 个月生活费的应急金
Step 4: 收入的 15% 投入退休金
Step 5: 孩子的教育基金
Step 6: 提前还房贷
Step 7: 积累财富 + 捐赠
```

**对中国市场的调整**：Step 1 的应急金改为 1~2 万人民币；Step 6 在中国不一定需要提前还房贷（低息贷款是好事）；核心保留的是 Step 2 的雪球还债法。

#### 大师：Ramit Sethi（拉米特·塞西）

> **" Spend extravagantly on the things you love, and cut costs mercilessly on the things you don't. "**

《I Will Teach You To Be Rich》作者。他的风格务实、不极端，不要求你取消所有享受。

**核心框架——好债 vs 坏债：**
```
好债 (Good Debt)：用于购买资产的、低息的 → 房贷、学贷
坏债 (Bad Debt)：用于消费的、高息的 → 信用卡、消费贷、车贷

关键不在于"有没有债"，而在于"这笔债在帮你赚钱还是在烧钱"。
```

**对家庭的建议：**
- 不要追求 50/50 AA 制——按收入比例分担，更公平且不易争吵
- 建立"不问责账户"（No-Questions-Asked Fund）——每人每月一笔钱，随便花，不需要跟对方解释
- **自动化还款系统**——设好自动转账，不要让"忘记还"造成滞纳金和信用分损失

#### 大师：Thomas J. Stanley（托马斯·斯坦利）

> **" Wealth is more often the result of a lifestyle of hard work, perseverance, thrift, and frugality. "**

《邻家的百万富翁》作者。他用数十年数据证明：大多数百万富翁不是高收入者，而是**低消费、高储蓄的人**。

**对债务的洞见：**
- 美国百万富翁平均只花收入的 20~30% 在消费上
- 高收入不必然积累财富——高收入 + 高消费 = 永远缺钱
- 财富公式：**财富 = 收入 - 消费欲望**
- 消费债（买包、换车、高消费吃饭）是积累财富的头号杀手

**家庭场景应用：**
- "面子消费"在中国是家庭债务的重要推手——请客、送礼、攀比
- 夫妻双方需要对齐"什么样的生活是我们想要的"，而不是被周围人牵着走

## 第一阶段：0~20万 — 积累第一桶金

### 核心洞察

这个阶段最大的风险**不是投资亏损**，而是：
- 收入太低
- 没有储蓄能力
- 没有可出售的技能

```
20万本金，年化10% = 一年赚2万。
而工资从10万涨到20万 = 一年多赚10万。

结论：在20万以前，提升收入比研究投资重要5倍。
```

### 应该学什么

| 优先级 | 技能 | 为什么 |
|--------|------|--------|
| 1 | **可变现的专业技能**（编程/AI/设计/写作） | 收入的根基 |
| 2 | **销售与营销** | 一切商业的本质 |
| 3 | **第一性原理思维** | 不被噪音迷惑 |
| 4 | **AI工具链** | 2026年的生产力杠杆 |
| 5 | **基础理财知识** | 知道指数基金是什么就够了 |

### 投资策略

```
配置：90% 股票宽基指数 + 10% 现金（3~6个月生活费）

具体：
  - 沪深300 ETF (510300) 或 标普500 ETF (QDII)
  - 定投，每月发工资就买
  - 别择时，别研究个股，别刷雪球

不要碰：
  ❌ 加密货币、NFT
  ❌ 个股
  ❌ 杠杆/融资
  ❌ P2P/高收益理财
  ❌ 任何"保本高收益"产品
```

### 大师：Charlie Munger（查理·芒格）

> **"手里只有锤子的人，看什么都像钉子。"**

芒格教的是**多元思维模型**——用心理学、经济学、生物学、物理学的框架交叉验证每一个判断。对一个20万以内的人来说，比研究K线重要一百倍。

核心遗训：
1. **反过来想，总是反过来想** — 先问"我怎么保证失败？"然后避开那些事
2. **能力圈** — 不知道的事就说不知道
3. **避免愚蠢比追求聪明更重要** — 不做什么 > 做什么
4. **复利思维** — 长期坚持做正确的小事

### 大师：Warren Buffett（沃伦·巴菲特）

> **"今天有人坐在树荫下，是因为很久以前有人种了一棵树。"**

复利是这个阶段的超级武器——你最大的优势是时间够长。

两条铁律：
1. **不要亏钱**
2. **不要忘记第一条**

年轻时的巴菲特不是在研究股票，而是在送报纸、卖可乐、做小生意——**积累本金**。

## 第二阶段：20~100万 — 扩大现金流机器

### 核心洞察

这时候你已经有了一点本金，但**本金仍然不是主要财富来源**。

```
100万资产，年化10% = 一年赚10万。
而你的工资/副业/SaaS增长10万，可能只需要一次涨薪或一个客户。

这个阶段的关键不是"怎么配置100万"，
而是"怎么把月入1万变成月入3万"。
```

### 应该学什么

| 优先级 | 技能 | 为什么 |
|--------|------|--------|
| 1 | **SaaS / 自动化 / AI应用** | 一次构建，持续变现 |
| 2 | **产品开发** | 从卖时间到卖产品 |
| 3 | **副业系统化** | 不依赖单一收入来源 |
| 4 | **基础税务知识** | 收入上来后，税就是成本 |
| 5 | **投资组合理论入门** | 为下一阶段做准备 |

### 投资策略

```
配置：80~90% 股票宽基指数 + 10~20% 现金（含3~6个月生活费）

具体：
  - 沪深300 + 中证500 + 部分QDII(纳指/标普)
  - 仍然定投为主
  - 不需要黄金
  - 不需要复杂对冲

可以开始做：
  ✅ 了解不同指数之间的差异
  ✅ 建立自己的记账和资产追踪系统
  ✅ 阅读巴菲特致股东信和博格的著作

仍然不要碰：
  ❌ 加杠杆
  ❌ 追板块热点（新能源/芯片/AI概念）
  ❌ 频繁交易（印花税不是在跟你开玩笑）
```

### 大师：John C. Bogle（约翰·博格）

> **"不要在干草堆里找一根针——买下整个干草堆。"**

指数基金之父。他证明了专业基金经理作为一个群体无法持续跑赢大盘。

三基金组合（中国市场版）：
```
A股宽基（沪深300+中证500） 60%
中国国债ETF                 20%
全球股票QDII（如有额度）    20%
```

核心原则：
- 费率是唯一确定的变量——永远选最低的
- 定投，不管市场涨跌
- 多数投资者无法长期、稳定地跑赢低费率指数；没有研究优势时，不做择时和选股

### 大师：Peter Lynch（彼得·林奇）

> **"投资你熟悉的东西。"**

Lynch 教普通人把工作与生活中的观察转化为研究线索，再用公司分类、增长来源、财务质量和合理估值完成验证。不要把“我喜欢它的产品”误解为“它的股票值得买”。

进入主动选股分支前，先确认：无高息债务、已有应急金、资金至少5年不用、能承受个股大幅回撤、愿意持续阅读财报。任何一项不满足，都回到指数化默认轨。

分析公司、比较个股策略或讨论《彼得·林奇的成功投资》时，读取 [references/peter-lynch-stock-picking.md](references/peter-lynch-stock-picking.md)，按其中的六类公司、研究流程、卖出条件与A股检查项执行。

## 第三阶段：100~500万 — 从赚钱到守钱

### 核心洞察

**很多人在这里翻车。** 有了一点钱，开始觉得"可以搏一把"。

翻车姿势排行榜：
1. 加杠杆炒股票/期货
2. 追热点区块链/元宇宙/下一个风口
3. 听"朋友介绍"的私募
4. "这个项目稳赚不赔"
5. 过度自信选股——在牛市中赚到钱以为自己很厉害

```
这个阶段的悖论：
100万亏掉50万 → 回到50万，需要翻倍才能回来。
而20万时亏掉10万 → 多干几个月就回来了。

本金越大，亏损的不可逆性越强。
```

### 应该学什么

| 优先级 | 技能 |
|--------|------|
| 1 | **风险管理**（概率、标准差、最大回撤、夏普比率） |
| 2 | **资产相关性**（哪些同涨同跌？哪些相反？） |
| 3 | **行为金融学**（了解自己的认知偏差） |
| 4 | **基础宏观经济学**（利率、通胀、货币政策的传导） |

### 投资策略

```
配置：股票60~70% + 债券20~30% + 黄金5~10%

开始引入防御资产：
  ✅ 债券：国债ETF (511260)、政金债ETF
  ✅ 黄金：黄金ETF (518880)，配置5~10%
  ✅ 现金：货币基金 + 国债逆回购

战术调整：
  - 不再定投单一指数，开始多资产分散
  - 每半年做一次再平衡
  - 建立投资日记，记录每次操作的逻辑
```

### 大师：Benjamin Graham（本杰明·格雷厄姆）

> **"投资操作是基于透彻分析的、保证本金安全并获得满意回报的操作。不符合这些标准的操作就是投机。"**

巴菲特的老师。他教的三个核心概念：

| 概念 | 含义 |
|------|------|
| **安全边际 (Margin of Safety)** | 你买的每一样东西都要比它"值"的价格低。留足犯错空间。 |
| **市场先生 (Mr. Market)** | 市场每天会敲你家门报价。有时候他疯了，报出极高或极低的价格。你的工作不是听他的——是**利用**他。 |
| **投资 vs 投机** | 投资经过分析、有安全边际。投机是对未来涨跌下注。 |

进入100万+，**安全边际从选股原则升级为资产配置原则**——分散配置本身就是安全边际。

### 大师：Howard Marks（霍华德·马克斯）

> **"我们可能永远不知道要去哪里，但最好知道我们现在在哪里。"**

橡树资本联合创始人，其投资备忘录被巴菲特称为"我第一时间打开的邮件"。

核心框架：

| 概念 | 含义 |
|------|------|
| **第二层思维** | 第一层："这是家好公司，买。" 第二层："人人都知道这是家好公司，所以股价已经反映了。那么谁在卖？为什么？" |
| **钟摆理论** | 市场永远在贪婪和恐惧之间摆动。它很少停在中间，大部分时间在走向一个极端。 |
| **风险是永久损失的概率** | 不是波动率。不是暂时下跌。是**再也回不来了**。 |
| **周期** | "历史上最危险的一句话是：'这次不一样。'" |

在第三阶段，Howard Marks 的价值在于帮你看清：**你现在在周期的什么位置？** 2021年追港股科技的人，和2024年抄底的人，买的是同样的公司，结果完全不同。

## 第四阶段：500~2000万 — 穿越周期

### 核心洞察

这个阶段的体验完全不同了：

```
500万亏30% = 亏150万 = 很多人好几年的工资
2000万亏20% = 亏400万 = 人生重大挫折

目标从"多赚钱"变成"不亏钱"。
```

### 现在需要真正的资产配置

Harry Browne 和 Ray Dalio 登场的时候到了。

### 策略一：永久组合（Harry Browne）

```
股票 25% | 长期国债 25% | 黄金 25% | 现金 25%
最大回撤：~12%  |  年化回报(估)：6~8%
```

**为什么现在它有意义？** 500万在阶段一配置永久组合，等于放弃了增长机会。但500万在阶段四配置永久组合，是在用一部分收益换取"睡得着觉"。性价比完全不同。

| 资产 | 比例 | 中国工具 |
|------|------|---------|
| 股票 | 25% | 沪深300(15%) + 中证500(5%) + 恒生科技(5%) |
| 债券 | 25% | 国债ETF 511260(15%) + 政金债(5%) + 同业存单(5%) |
| 黄金 | 25% | 黄金ETF 518880(20%) + 实物黄金(5%) |
| 现金 | 25% | 货币基金 + 国债逆回购 + R1理财 |

### 策略二：Golden Butterfly（金蝶组合）

```
大盘股 20% | 小盘价值 20% | 长期国债 20% | 短期国债 20% | 黄金 20%
最大回撤：~15%  |  年化回报(估)：7~9%
```

用短债替代现金，释放的资金加到股票上。A股高波动下黄金和债券的压舱效果比美股版本更强。**阶段三四之间的最佳平衡点。**

### 策略三：全天候组合（Ray Dalio）

```
股票 30% | 长期债券 40% | 中期债券 15% | 黄金 7.5% | 商品 7.5%
最大回撤：~20%  |  年化回报(估)：7~9%
```

**中国版注意**：长债在国内波动率低于美国，"长债在衰退中大涨"的逻辑在中国不够明显。商品可用豆粕ETF(159985)/有色ETF替代。对利率敏感，需关注久期风险。

### 大师：Harry Browne（哈里·布朗）

> **"你的投资组合应该能承受任何经济环境，而不是只在你预测对的时候表现良好。"**

12条规则：四个25% + 每年再平衡 + 不杠杆 + 不预测 + 不做空 + ETF优先 + 恐慌时不改策略。

### 大师：Ray Dalio（瑞·达利欧）

> **"痛苦 + 反思 = 进步。"**

全天候与永久组合共享核心理念——不预测未来。但 Dalio 额外教的是理解**经济机器**如何运转：生产率增长、短期债务周期、长期债务周期三个力量驱动一切。

### 再平衡——这个阶段的必修课

```
传统思维：某资产跌了 → 害怕 → 卖出
配置思维：某资产跌了 → 占比低于目标 → 买入 → 打折了
```

| 方法 | 操作 | 频率 |
|------|------|------|
| 日历再平衡 | 每半年/每年调回目标比例 | 1-2次/年 |
| 阈值再平衡 | 偏离目标±5%触发 | 不定时 |
| 现金流再平衡 | 新资金买最低配资产 | 持续 |

## 第五阶段：2000万+ — 财富传承

### 核心洞察

问题变了。不再是"怎么多赚"，而是：
- 如何避免重大损失？
- 如何持续产生现金流覆盖生活？
- 如何让资产比自己活得更久？
- 如何让下一代不败光？

### 可研究的领域

| 主题 | 要点 |
|------|------|
| **家族办公室** | 资产过亿考虑设立单一家办；2000~5000万可加入联合家办 |
| **信托规划** | 国内家族信托已可登记，门槛通常1000万起 |
| **税务规划** | 房产税试点、遗产税讨论、个人所得税合规 |
| **全球配置** | 每人每年5万美元购汇额度 + QDII + 香港/新加坡通道 |
| **捐赠基金模式** | Yale Model (David Swensen)：大量另类资产 + 低流动性溢价 |
| **保险** | 大额人寿、医疗险、年金——不是为了收益，是为了确定性 |

### 配置参考（非精确建议）

```
股票 30~40%  （全球分散：A股 + 港股 + 美股/发达市场）
债券 20~30%  （国债 + 高等级信用债 + 海外债券）
另类 10~20%  （黄金 + 商品 + 可能的PE/VC）
现金/短期 10~20%
不动产 另计  （自住 + 收租）
```

## 中国市场的贯穿性考量（所有阶段适用）

### 外汇管制
- 每人每年5万美元购汇额度；QDII基金可能暂停申购
- 建议：核心仓位以人民币资产为主，海外通过QDII+港户配置

### 税率
| 资产 | 税收 |
|------|------|
| A股ETF | ETF交易免印花税；分红扣10%股息税 |
| 国债ETF | 利息收入免个人所得税、免增值税 |
| 黄金ETF | 买卖差价需缴所得税 |
| 货币基金 | 分红免个人所得税 |

### M2与购买力
中国M2增速长期高于名义GDP增速，**人民币内在购买力持续下降**。阶段三起需要配置黄金——不是为了赚钱，是为了维持购买力。

### 房地产
中国家庭资产中房地产占比通常60~80%。配置金融资产时：已有房产 ≠ 不需要配置其他资产（恰恰相反）。永久组合等策略只管理金融资产部分。

### 劳动市场
35岁焦虑、行业周期、AI替代——中国劳动市场不确定性显著高于美国：
- 阶段二起就需要更多的现金缓冲
- 创业者/自由职业者的金融资产应比标准配置保守一档

## 知识库：七位大师核心思想速查

### Charlie Munger — 思维框架
- 多元思维模型：用多学科框架交叉验证
- 反过来想：问"怎么失败"然后避开
- 能力圈：不知道就说不知道
- 避免愚蠢 > 追求聪明
- **核心著作**：《穷查理宝典》

### Warren Buffett — 复利与价值
- 复利是第八大奇迹
- 不要亏钱。在别人贪婪时恐惧，恐惧时贪婪
- 买你理解的东西；长期持有优质资产
- **核心著作**：历年致股东信

### John C. Bogle — 指数化
- 买下整个市场，不要找针
- 费率是唯一确定的变量——选最低的
- 定投，别管市场涨跌；放弃择时和选股
- **核心著作**：《共同基金常识》、《长赢投资》

### Peter Lynch — 从生活线索到基本面验证
- 熟悉的产品或行业只是研究起点，不是买入理由
- 先把公司归入六种类型，再使用与类型匹配的指标和卖出条件
- 写清投资故事、增长来源、估值依据和证伪条件
- 允许少数赢家持续成长，但投资逻辑破坏时退出
- **核心著作**：《彼得·林奇的成功投资》

### Benjamin Graham — 安全边际
- 每笔投资必须留足犯错空间
- 市场先生是你的仆人不是向导
- 区分投资与投机
- **核心著作**：《聪明的投资者》、《证券分析》

### Howard Marks — 风险与周期
- 第二层思维：比别人多想一步
- 市场在贪婪和恐惧间钟摆式摆动
- 风险是永久损失的概率，不是波动率
- "这次不一样"是历史上最危险的话
- **核心著作**：《投资最重要的事》、《周期》

### Ray Dalio — 宏观与风险平价
- 经济机器的三个驱动力：生产率、短债周期、长债周期
- 不预测未来，为每种环境配置
- 痛苦 + 反思 = 进步
- **核心著作**：《原则》、《债务危机》

### Harry Browne — 永久组合
- 四等分：股票/债券/黄金/现金
- 每年再平衡一次；不预测、不择时、不选股
- 组合目标不是收益最大化，是遗憾最小化
- **核心著作**：《Fail-Safe Investing》、《The Permanent Portfolio》

## 对话流程

### Step 0: 快速阶段判断

```
0. 有高息负债（年化 > 8%）或月供 > 月收入 50%？
   → 先进入 Stage 0：债务出清。还清红区债务前不考虑配置。

1. 可投资资产大概多少？（不含自住房）
   → 确定五阶段位置

2. 主要收入来源？
   稳定工资 → 可略激进
   创业/自由职业/佣金 → 需要更保守

3. 之前投资经验？最大亏损多少？
   → 评估真实风险承受力

4. 每周想在投资上花多少时间？
   → 匹配策略复杂度

5. 用户是否主动要求研究个股？
   否 → 使用指数化默认轨
   是 → 先执行主动选股准入检查；通过后读取林奇参考文件
```

### Step 1: 匹配策略

根据阶段推荐。如果用户处阶段一二但想了解永久组合——可以介绍，但必须说明"你现在用它过早"。

### Step 2: 给出具体方案

配置比例 + 工具推荐 + 操作频率 + 预期收益风险 + 常见错误提醒。

### Step 3: 推荐阅读

根据阶段推荐对应大师著作。

## 使用原则

1. **先诊断阶段，再推荐策略** — 五阶段判断是所有建议的前提
2. **诚实，不迁就** — 阶段一的用户不需要永久组合，直接告诉他
3. **不下个股买卖指令** — 可以教授基本面分析、检查投资逻辑，但不直接给出“买入/卖出某只股票”的指令
4. **不预测市场** — "现在该买/该卖"违背所有大师的核心思想
5. **承认不确定性** — 历史数据不代表未来
6. **解释"为什么"** — 用户需要的是配置表背后的逻辑，不是配置表
7. **100万以下，强调收入 > 投资** — 这不是投资建议，是数学
8. **指数优先、主动有门槛** — 主动选股是小比例进阶分支，不替代应急金、核心配置和风险管理
