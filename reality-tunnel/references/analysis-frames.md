# 深度分析框架

仅在问题复杂、高风险、强冲突或用户要求深入分析时加载本文件。按问题需要选用，不要把所有清单机械塞进输出。

## 事实账本

| 类型 | 判断标准 | 常见错误 |
|---|---|---|
| 观察事实 | 可指出观察、记录、数据或可信来源 | 把转述、感受或结论写成事实 |
| 解释 | 对意义、模式或原因的归纳 | 省略推理链，伪装成客观描述 |
| 预测 | 在未来条件下可被检验 | 使用“必然”“永远”逃避条件 |
| 价值 | 对好坏、公平、风险、优先级的偏好 | 把偏好包装成自然规律 |
| 未知 | 缺证据且可能改变决策 | 用最顺手的故事填空 |

先问：如果只能验证一项未知，哪一项最可能改变行动？

## 信念拆解

按以下顺序压缩一个观点：

1. **主张**：它具体断言什么？
2. **范围**：对谁、何时、在什么条件下成立？
3. **机制**：原因如何产生结果？中间环节是什么？
4. **证据**：支持来自观察、样本、权威、类比还是感受？
5. **反证**：什么观察会降低置信度？
6. **替代解释**：同一现象还有哪些不同机制？
7. **行动含义**：相信它会让人做什么或停止做什么？

把“我找不到反例”与“没有反例”分开。

## 隐藏假设分类

- **范围偷渡**：从部分样本推到所有人、所有时间或所有情境。
- **永久性**：把当前状态当作稳定趋势或终局。
- **必然因果**：把相关、先后或共同变化当成单一因果。
- **静态系统**：忽略参与者会学习、适应、套利或改变规则。
- **零和预设**：默认一方获益必然要求另一方受损。
- **动机读心**：从行为直接推断他人意图或品格。
- **身份固定**：把“现在做不到”改写成“我就是做不到的人”。
- **单一指标**：让易测量指标取代真正目标。
- **遗漏基准率**：只看鲜明案例，不看总体发生率或选择偏差。
- **控制错位**：把不可控变量当责任，或把可控变量当宿命。

只报告对结论有杠杆的假设。为每项写出：“若为假，什么结论会变化？”

## 替代模型生成矩阵

从不同维度生成候选，再选择机制差异最大的三个到五个：

- **层级**：个体能力、互动关系、组织流程、市场结构、文化叙事。
- **时间**：短期冲击、适应期、长期复利或反身性反馈。
- **激励**：公开目标、实际奖励、风险转移、信息不对称。
- **选择**：主动选择、受限选择、路径依赖、默认项效应。
- **测量**：真实变化、口径变化、样本偏差、代理指标失真。
- **多因**：必要条件、促成条件、触发条件是否被混为一谈。

拒绝“也许一切都有可能”的空模型。一个可用模型必须产生与其他模型不同的观察预测或行动建议。

## 模型比较

使用以下字段，不必强行打分：

| 模型 | 解释力 | 独特预测 | 关键证据 | 忽略项 | 错判代价 | 行动含义 | 置信度 |
|---|---|---|---|---|---|---|---|

置信度语言：

- **较高**：有多条独立证据，机制与预测相符，主要替代解释较弱。
- **中等**：机制合理且有部分证据，但关键变量未验证。
- **较低**：主要依赖类比、少量案例、动机推测或不可证伪叙事。

## 区分性验证

优先设计能让模型产生不同结果的测试：

1. 写出每个模型在同一条件下预测的可观察差异。
2. 选择成本最低、反馈最快、伤害最小的观察或实验。
3. 预先写明支持、削弱和无法判断的阈值。
4. 防止测试被事后重新解释成“无论结果都证明我对”。

## 冲突模式

不要先判断谁“认知有问题”。按以下结构处理：

1. **共享事实**：双方都承认的事件和约束。
2. **争议事实**：可通过记录、数据或第三方信息核实的分歧。
3. **语义分歧**：同一词是否指不同标准，如“完成”“质量”“尊重”。
4. **价值分歧**：速度、稳定、控制、公平、面子、风险的优先级差异。
5. **激励分歧**：谁获得收益、承担代价或被什么指标评价。
6. **双方现实隧道**：各自看到了什么、忽略了什么、在防御什么风险。
7. **共同测试**：约定一项可观察结果，而非继续争夺动机解释权。

当证据明显不对称、存在欺骗或伤害时，明确指出；多视角不要求虚假平衡。

## 思维调试表达

仅把模式当作当前推理策略：

| 可能模式 | 文本迹象 | 更有用的重写 |
|---|---|---|
| 灾难化 | 从风险直接跳到最坏终局 | 最坏、最可能、可恢复结果分别是什么？ |
| 二元化 | 只有成功/失败、支持/反对 | 是否存在阈值、阶段或部分成功？ |
| 过度概括 | 一次事件变成“总是如此” | 样本范围和反例是什么？ |
| 读心 | 未经证据断言他人动机 | 可观察行为与动机解释分别是什么？ |
| 身份绑定 | 能力判断变成固定自我定义 | 当前技能、情境限制和可训练部分是什么？ |
| 沉没成本 | 因已投入而继续投入 | 从今天重新选择，还会投入吗？ |
| 确定性膨胀 | 用绝对词掩盖概率与条件 | 在什么条件下、以多大置信度成立？ |
| 模型实在化 | 把分类或指标当成对象本身 | 这个标签遗漏了哪些实际变化？ |

输出“检测到”之前必须引用具体语言迹象；证据弱时使用“可能”。

## 行动组合

- **低后悔行动**：多个合理模型下都有收益。
- **信息行动**：首要目标是缩小关键未知，而非立刻赢得结果。
- **条件行动**：只有观察到预定信号才执行。
- **退出条件**：何时停止继续投入或停止争论。
- **复盘点**：何时用新证据重算模型权重。

对重大决定优先采用可逆、小规模、分阶段承诺，除非延迟本身的代价更高。
