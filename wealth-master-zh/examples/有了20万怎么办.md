## 阶段定位

用户已积累20万本金，进入**阶段二（20~100万）**。

```
20万 × 10% = 一年赚2万
月薪涨3000 = 一年多赚3.6万

20万阶段，收入增长仍然比投资收益重要。
但你已经有本金了，可以开始让钱也干活。
```

### 已有20万怎么放

| 资金用途 | 操作 |
|----------|------|
| 3年内用不上（首选） | 一次性按比例买入，后续每月定投维持。别等"跌了再买"——你不择时 |
| 3年内可能用（买房/结婚等） | 10万一次性买入 + 10万货币基金，保持灵活性 |

## 优先级排序

```
1. 主业涨薪/跳槽 ← 你最大的杠杆
2. 保持每月定投 ← 让习惯自动运行
3. 学一个副业技能（SaaS/AI工具/写作） ← 扩大收入来源
4. 别花时间研究股票 ← 你的时间比超额收益值钱得多
```

## 常见疑问

| 问题 | 回答 |
|------|------|
| 要不要配置黄金？ | 不需要。黄金在阶段二的作用是防大通胀，20万本金扛不住大通胀是正常的——应对方式是提高收入，不是买黄金 |
| 要不要买债券？ | 不需要。债券控制回撤。20万跌30%亏6万，工资几个月就赚回来。没必要用债券牺牲收益 |

## 一句话总结

> 20万阶段就做两件事：**定投宽基，提升收入。** 其他都是噪音。

等到了100万再来，那时候才需要开始考虑防御。
