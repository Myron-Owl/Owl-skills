# 2D100 强/弱/未命中系统设计分析

**输入：**
```
Analyze: 战棋RPG战斗系统的2D100命中判定
  - 基础命中率 = (攻击-防御)/100，钳制5%~95%
  - 投2个D100：2个通过→强命中(max伤)，1个通过→弱命中(随机伤)，0个通过→Miss
  - 武器伤害范围：斧头15-55(宽范围)，剑30-40(窄范围)
Mode: breakdown
Depth: high
Lens: auto
```

**关键发现：**
1. 实际命中率 = 2p-p²（p=70%显示→实际91%），面板与真实值有偏差
2. 武器分化设计有效——宽范围武器在高命中下优势显著扩大
3. Miss率大幅下降（70%→9%），减少了Roguelike叙事的戏剧性失败

**分析视角：Miyamoto Lens × Sid Meier Lens × Miyazaki Lens**
