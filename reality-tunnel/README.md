# Reality Tunnel Analyzer（现实隧道分析器）

一个面向 Codex 的认知决策 Skill。受 Robert Anton Wilson《Prometheus Rising》中 Reality Tunnel 与 meta-programming 思想启发，但不把书籍内容做成知识库，而是将其转化为可执行的分析流程。

它用于区分事实与解释、发现隐藏假设、生成竞争模型，并把认知重构落到可验证行动。

## 适用场景

- 拆解绝对化判断和强信念
- 辅助复杂决策，发现被当前框架排除的选项
- 复盘团队冲突中的事实、价值与激励分歧
- 检查灾难化、二元化、身份绑定等当前推理模式
- 分析设计者与玩家对同一游戏机制的不同理解

## 调用方式

在 Codex 中显式调用最可靠：

```text
使用 $reality-tunnel 分析“35 岁以后普通人没有机会了”。
```

复杂决策：

```text
使用 $reality-tunnel 深度分析我是否应该离开当前项目。
区分事实、解释和未知项，生成至少三个竞争模型，
并给出低后悔行动、验证实验和退出条件。
```

游戏设计：

```text
使用 $reality-tunnel 的玩家现实隧道模式分析疲劳机制。
比较设计者、新手、核心玩家和时间有限玩家的解释，
定位意图与体验的错位，并设计验证实验。
```

团队冲突：

```text
使用 $reality-tunnel 分析策划与程序关于品质和排期的冲突。
区分共享事实、争议事实、价值分歧、激励分歧和双方现实隧道。
```

## 核心流程

```text
定义真正的问题
  → 建立事实账本
  → 描述当前现实隧道
  → 暴露关键隐藏假设
  → 生成至少三个替代模型
  → 比较模型并设计区分性验证
  → 转化为可逆实验与行动选项
```

## 输出内容

- 暂定结论及适用条件
- 事实、解释、预测、价值和未知项
- 当前现实隧道及成立条件
- 会改变结论的高杠杆假设
- 不同机制的替代模型
- 能区分模型的证据或小实验
- 低后悔行动、条件行动和停止条件
- 有文本依据时的思维调试

## 游戏设计扩展

游戏问题会按以下感知链路分析：

```text
客观规则
  → 可见信息与反馈
  → 玩家因果解释
  → 情绪与价值判断
  → 行为选择
  → 社区叙事
```

重点检查能动性、可读性、公平感、胜任感、时间尊重、选择意义、收益契约和身份社交影响。

## 边界

- 不把任何现实隧道宣称为绝对真相。
- 多视角不意味着所有模型证据权重相同。
- 不根据一次表达进行人格或心理疾病诊断。
- 不用廉价乐观替换悲观模型，也不用不可证伪叙事回避证据。
- 健康、法律和财务等高风险问题仍需相应专业意见。

## 安装

将整个目录复制到 Codex 技能目录：

```text
~/.codex/skills/reality-tunnel/
```

重新打开 Codex 任务后，使用 `$reality-tunnel` 调用。
