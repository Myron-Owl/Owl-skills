# 毛泽东思想分析框架 (Mao Thought)

> 一个 Claude Code skill — 将毛泽东思想方法论转化为现代问题的系统化分析工具。

## 概述

这个 skill 让 Claude 使用**毛泽东思想的分析方法论**来解决问题。它不是历史教育工具，而是将实事求是、矛盾论、实践论、群众路线、持久战、集中优势兵力等方法论，翻译成现代场景下可操作的分析框架：

1. **实事求是** — 搞清客观事实，区分事实与假设，识别信息盲区
2. **矛盾分析** — 识别主要矛盾和矛盾的主要方面，找到核心突破口
3. **持久战分解** — 将大目标分解为阶段化推进计划
4. **集中优势兵力** — 聚焦唯一重点，明确"不做清单"
5. **群众路线** — 建立反馈闭环，从群众中来到群众中去

适用于：产品设计、职业规划、商业策略、项目推进、个人成长——任何需要深度分析的领域。

## 使用方法

在 Claude Code 中调用：

```
/mao-thought 我想分析...[你的问题]
```

自然语言也可以触发：
- "帮我用毛泽东思想分析一下..."
- "客观分析这个问题的核心矛盾..."
- "怎么找到这件事的主要矛盾？"

## 方法论来源

所有分析方法论均源自毛泽东思想的核心概念——《矛盾论》《实践论》《论持久战》《反对本本主义》等经典著作中的方法论工具，剥离其特定历史语境，保留其作为分析工具的逻辑内核。

## 使用原则

1. **以分析为主，引用为辅** — 引用原文支持分析，非展示知识
2. **用现代语言翻译** — 把"群众"翻译成"用户/客户/团队"
3. **结论要有操作性** — 分析完必须有"下一步做什么"
4. **承认局限性** — 不适合的问题不强套
5. **不搞个人崇拜** — 框架的价值在于逻辑，不在于出处
