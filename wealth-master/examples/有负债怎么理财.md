## 用户画像

| 项目 | 本人 | 配偶 |
|------|------|------|
| 债务类型 | 低息房贷（~3~4%） | 高息消费贷（8~24%） |
| 月供/收入 | < 20% | 300%+ → 下月79% → 明年<30% |

## 分析诊断

### 本人：绿区债务（房贷）

```
月供占收入 < 20%，利率远低于8%
→ 不需要提前还
→ 通胀在帮你稀释
→ 正常进行你的阶段配置
```

### 配偶：红区债务（消费贷）

月供 300%+ 说明在借新还旧滚雪球。这是财务紧急状态——在五阶段之前，需要先进入 **Stage 0：债务出清**。

## 行动方案

### 你（低息房贷）— 按节奏走

- 正常还月供，不影响生活
- 多余的钱按自身阶段正常定投
- 不要帮配偶还消费贷——会掩盖问题

### 配偶（高息消费贷）— Stage 0 债务出清

```
优先级 1：立刻停止新增消费贷
优先级 2：砍掉所有非必要支出（外卖/打车/购物暂停3个月）
优先级 3：雪球法或雪崩法还债

下个月降到 79% → 说明还款计划在起作用
明年降到 30% → 恢复正常，可以开始积累
```

### 家庭层面

| 原则 | 具体做法 |
|------|---------|
| 透明 | 两人坐下来，把各自债务列总表：利率、月供、剩余期数 |
| 分开处理 | 房贷继续共同还；消费贷由借款人承担还款责任 |
| 救急不救懒 | 可以短期帮还，但前提是对方承诺不再新增消费贷 |
| 先救婚姻 | 每天为钱吵架就先把沟通问题解决，再谈债务 |

## 关键数字

```
消费贷年化 15%
→ 还掉它 = 无风险年化 15% 收益
→ 巴菲特长期年化约 20%
→ 没有任何投资能保证 15%

数学结论：还清前不要投资。
```

## 一句话总结

> 你的房贷不是问题，她的消费贷才是头号敌人。**红区债务还清之前，不要谈配置。**

等消费贷降到月收入 30% 以下后再来做家庭资产配置。
